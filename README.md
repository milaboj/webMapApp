# Covid-19 Prevalence in the United States by County in 2020

### Introduction
This project looks at how different counties and states were impacted by Covid-19 in 2020 by looking both at Covid-19 rates per 1,000 residents and total cases. The [first map](https://milaboj.github.io/webMapApp/map1.html) is a choropleth map which shows Covid-19 rates per 1,000 residents per county seen below. ![here](/img/screenshot1.png)

The [second map](https://milaboj.github.io/webMapApp/map2.html) is a proportional symbol map which shows total Covid-19 cases per county seen below. ![here](/img/screenshot2.png)


#### Primary Functions
Interactive choropleth map and proportional symbol maps were created with legends, pop-ups, and introductory panels.
